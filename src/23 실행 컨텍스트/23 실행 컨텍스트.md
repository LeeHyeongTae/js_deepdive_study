# 23. 실행 컨텍스트

- 자바스크립트의 동작 원리를 담고 있는 핵심 개념.
- 스코프를 기반으로 식별자와 식별자에 바인딩된 값을 관리하는 방식.
- 호이스팅이 발생하는 이유
- 클로저의 동작 방식
- 태스크 큐와 함께 동작하는 이벤트 핸들러
- 비동기 처리의 동작 방식

## 23-1. 소스코드 타입

- ECMAScript는 소스코드를 4가지 타입으로 구분. 소스코드는 실행 컨텍스트를 생성.


| 소스코드의 타입 | 설명 |
|:-----------:|:------------------------------------------------------------:|
|global code|전역에 존재하는 소스코드, 전역에 정의된 함수, 클래스 등의 내부 코드는 포함되지 않음.|
